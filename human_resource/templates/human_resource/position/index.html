Position Helper Guide

A position represents a job role in a specific department. Examples include: Accountant, Nurse, Lab Technician.

NAME: The job title. Should be unique within the same department (e.g. only one "Manager" per department).

DEPARTMENT: Select the department this position belongs to. Each department can have multiple positions.

LEVELS: Associate one or more job levels with this position. These define the rank or hierarchy for staff holding the position.

Tip: Use levels to differentiate roles like "Senior Nurse" vs "Junior Nurse".
