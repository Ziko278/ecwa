Specialization Helper

Specializations are used to define the different medical fields of expertise available. Each specialization has a name, a unique code, and a base consultation fee.

A group allows you to link several specializations together. For example, "Surgical" could be a group for Orthopedics, General Surgery, and Neurosurgery.

Tip: The code is automatically generated from the first three letters of the name if not provided.
Note: The base consultation fee is the default price for a consultation in this specialization.
