Consultation Room Helper Guide

A consultation room is a specific location within a building block where consultations take place. Each room is linked to a block and can be assigned a primary medical specialization.

NAME: The name of the room (e.g., Room 101, B2).

BLOCK: Select the consultation block this room is located in.

SPECIALIZATION: Assign a primary specialization. This helps in directing doctors and patients to the correct room. This is optional.

Tip: The capacity field can be used to indicate how many doctors can use the room simultaneously.
